## اولین برنامه پایتون شما

بیایید با برنامه کلاسیک "Hello, World!" شروع کنیم:

```python
print("Hello, World!")
```

**این چه کاری انجام می‌دهد:** متن "Hello, World!" را روی صفحه نمایش می‌دهد.

خودتان امتحان کنید! مفسر پایتون را باز کنید یا یک فایل `.py` بسازید و آن را اجرا کنید.

---

## قانون خاص پایتون: تورفتگی مهم است

پایتون به دلیل استفاده از **تورفتگی** (فضاهای در ابتدای خطوط) برای سازماندهی کد، منحصربه‌فرد است.

### مثال خوب (تورفتگی صحیح)

```python
if age >= 18:
    print("You are an adult")
    print("You can vote")
```

### مثال بد (تورفتگی اشتباه)

```python
if age >= 18:
print("You are an adult")      # ❌ اشتباه! فاقد فاصله
    print("You can vote")       # ❌ اشتباه! فاصله ناسازگار
```

### قانون
- برای هر سطح تورفتگی از **۴ فاصله** استفاده کنید (بیشتر ویرایشگرها این کار را به صورت خودکار انجام می‌دهند)
- همیشه در فاصله‌گذاری خود سازگار باشید

## کامنت‌ها: یادداشت‌ها برای انسان‌ها

کامنت‌ها متنی هستند که پایتون آنها را نادیده می‌گیرد. از آنها برای توضیح کد خود استفاده کنید.

```python
# این یک کامنت است - پایتون این خط را نادیده می‌گیرد
print("Hello")  # این کامنت در انتهای خط است

# می‌توانید چندین کامنت داشته باشید
# مثل این
# برای توضیح چیزی طولانی‌تر
```

**چرا از کامنت استفاده کنیم؟**
- برای توضیح اینکه کد چه کاری انجام می‌دهد
- برای یادآوری به خودتان بعداً
- برای غیرفعال کردن موقت کد بدون حذف آن

---

## متغیرها: ذخیره داده

متغیرها مثل **جعبه‌های برچسب‌دار** هستند که اطلاعات را در آنها ذخیره می‌کنید.

### ایجاد متغیرها

```python
name = "Alice"      # یک جعبه برچسب‌دار "name" حاوی "Alice"
age = 25            # یک جعبه برچسب‌دار "age" حاوی ۲۵
height = 1.65       # یک جعبه برچسب‌دار "height" حاوی ۱.۶۵
is_student = True   # یک جعبه برچسب‌دار "is_student" حاوی True
```

### قوانین نام‌گذاری متغیرها

**✅ نام‌های خوب:**
```python
user_name = "Bob"
total_count = 100
favorite_color = "blue"
```

**❌ نام‌های نامعتبر (باعث خطا می‌شوند):**
```python
2nd_place = "silver"      # نمی‌تواند با عدد شروع شود
my-name = "Alice"         # نمی‌تواند از خط تیره استفاده کند
my name = "Bob"           # نمی‌تواند از فاصله استفاده کند
class = "Python"          # نمی‌تواند از کلمات رزرو شده استفاده کند
```

### نکات برای نام‌های خوب
- از نام‌های توصیفی استفاده کنید (`total_price` بهتر از `tp` است)
- از حروف کوچک با زیرخط استفاده کنید (`snake_case`)
- سازگار باشید

## انواع داده: انواع مختلف اطلاعات

### ۱. رشته‌ها (متن)

```python
# از تک‌نقل‌قول یا دو نقل‌قول استفاده کنید
greeting = "Hello"

# هر دو یک کار را انجام می‌دهند!
message = "It's a nice day"
quote = 'He said "Hello"'
```

### ۲. اعداد

```python
# اعداد صحیح (اعداد کامل)

year = 2024

# اعداد اعشاری (اعداد با ممیز)
price = 19.99
pi = 3.14159
```

### ۳. بولی (درست/نادرست)

```python
is_sunny = True
is_raining = False
```

## دریافت ورودی از کاربران

تابع `input()` به برنامه شما اجازه می‌دهد از کاربر اطلاعات بگیرد:

```python
name = input("What's your name? ")
print(f"Hello, {name}!")
```

**مهم:** `input()` همیشه به شما متن (یک رشته) می‌دهد. اگر می‌خواهید اعداد:

```python
# تبدیل رشته به عدد صحیح
age = int(input("How old are you? "))

# تبدیل رشته به عدد اعشاری
height = float(input("Your height in meters: "))
```

## اشتباهات رایج مبتدیان

### اشتباه ۱: نقل‌قول‌های ناهمخوان

```python
# ❌ اشتباه
print("Hello')
print('Hello")

# ✅ صحیح
print("Hello")
print('Hello')
```

### اشتباه ۲: تورفتگی اشتباه

```python
# ❌ اشتباه
if age > 18:
print("Adult")

# ✅ صحیح
if age > 18:
    print("Adult")
```

### اشتباه ۳: استفاده از = به جای == برای مقایسه

```python
# ❌ اشتباه (= برای مقداردهی است)
if x = 5:
    print("x is 5")

# ✅ صحیح (== برای مقایسه است)
if x == 5:
    print("x is 5")
```

### اشتباه ۴: فراموش کردن پرانتز

```python
# ❌ اشتباه
print "Hello"

# ✅ صحیح
print("Hello")
```

### اشتباه ۵: حساسیت به بزرگی و کوچکی حروف

```python
# ❌ اشتباه
Print("Hello")
PRINT("Hello")

# ✅ صحیح
print("Hello")
```

پایتون به بزرگی و کوچکی حروف حساس است! `print` و `Print` متفاوت هستند.

## خودتان امتحان کنید: تمرین‌ها

### تمرین ۱: سلام ساده

یک برنامه بسازید که:
۱. نام کاربر را بپرسد
۲. سن کاربر را بپرسد
۳. یک سلام با هر دو اطلاعات چاپ کند

```python
name = input("What's your name? ")
age = input("How old are you? ")
print(f"Hello {name}! You are {age} years old.")
```

### تمرین ۲: ماشین‌حساب ساده

یک برنامه بسازید که:
۱. دو عدد بپرسد
۲. آنها را با هم جمع کند
۳. نتیجه را نشان دهد

```python
num1 = int(input("Enter first number: "))
num2 = int(input("Enter second number: "))
result = num1 + num2
print(f"The sum is: {result}")
```

### تمرین ۳: رفع خطاها

این برنامه ۳ خطا دارد. آیا می‌توانید آنها را پیدا و رفع کنید؟

```python
# اصلی (با خطاها)
name = input("What's your name? )
Age = input("How old are you? )
print("Hello " + name + you are " + age + " years old")
```

## نکات کلیدی

۱. **پایتون قابل خواندن است** - تقریباً شبیه انگلیسی به نظر می‌رسد
۲. **تورفتگی مهم است** - از ۴ فاصله برای بلوک‌های کد استفاده کنید
۳. **متغیرها داده ذخیره می‌کنند** - مثل جعبه‌های برچسب‌دار
۴. **انواع داده مختلف** - متن، اعداد، درست/نادرست
۵. **ورودی/خروجی** - از `input()` و `print()` برای ارتباط استفاده کنید
۶. **کامنت‌ها کد را توضیح می‌دهند** - با `#` شروع می‌شوند
