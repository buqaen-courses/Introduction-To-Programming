# محیط توسعه پایتون: ابزارهای کدنویسی

## آنچه خواهید آموخت
- چه ابزارهایی برای نوشتن کد پایتون نیاز دارید
- چگونه یک ویرایشگر کد انتخاب کنید
- چگونه برنامه‌های پایتون را اجرا کنید

---

## برای شروع کدنویسی چه چیزهایی نیاز دارید

برای نوشتن و اجرای برنامه‌های پایتون، نیاز دارید به:

۱. **پایتون** روی کامپیوتر خود نصب شده (مفسر)
۲. یک **ویرایشگر کد** (جایی که کد را می‌نویسید)
۳. یک **ترمینال/خط فرمان** (جایی که کد را اجرا می‌کنید)

همین! بیایید به هر کدام نگاه کنیم.

---

## ۱. نصب پایتون

### بررسی اینکه آیا پایتون قبلاً نصب شده

ترمینال/خط فرمان خود را باز کنید و تایپ کنید:

```bash
python --version
```

یا در برخی سیستم‌ها:

```bash
python3 --version
```

اگر چیزی مثل `Python 3.9.7` را دیدید، همه چیز آماده است!

### نصب پایتون (اگر ندارید)

#### ویندوز
۱. به [python.org/downloads](https://python.org/downloads) بروید
۲. روی "Download Python" (دکمه بزرگ زرد) کلیک کنید
۳. نصب‌کننده را اجرا کنید
۴. **مهم:** طی نصب، کادر "Add Python to PATH" را علامت بزنید
۵. روی "Install Now" کلیک کنید

#### مک
۱. Terminal را باز کنید
۲. Homebrew را نصب کنید (اگر ندارید): به [brew.sh](https://brew.sh) مراجعه کنید
۳. تایپ کنید: `brew install python3`

#### لینوکس
پایتون معمولاً از قبل نصب شده. اگر نه:

```bash
# اوبونتو/دبیان
sudo apt update
sudo apt install python3

# تأیید نصب
python3 --version
```

## ۲. انتخاب یک ویرایشگر کد

یک ویرایشگر کد مثل یک پردازشگر متن برای برنامه‌نویسی است. در اینجا چند گزینه خوب وجود دارد:

### برای مبتدیان: IDLE (همراه پایتون می‌آید)

**IDLE** هنگام نصب پایتون شامل می‌شود. ساده و کامل برای یادگیری است.

**نحوه باز کردن IDLE:**
- **ویندوز:** IDLE را در منوی Start جستجو کنید
- **مک:** در Applications → Python پیدا کنید
- **لینوکس:** در ترمینال `idle` را تایپ کنید

**ویژگی‌ها:**
- رابط ساده
- نمایش کد با رنگ (برجسته‌سازی نحو)
- راهنمای داخلی برای مبتدیان

### برای کاربران متوسط: VS Code

**VS Code** (Visual Studio Code) رایگان، قدرتمند و محبوب بین برنامه‌نویسان است.

**نحوه دریافت:**
۱. از [code.visualstudio.com](https://code.visualstudio.com) دانلود کنید
۲. آن را نصب کنید
۳. افزونه Python را نصب کنید (VS Code این را پیشنهاد می‌کند)

**چرا VS Code؟**
- رایگان و روی همه پلتفرم‌ها کار می‌کند
- با پیشنهادها به شما کمک می‌کند کد بهتری بنویسید
- ترمینال داخلی دارد

### گزینه‌های دیگر

| ویرایشگر | بهترین برای | نحوه دریافت |
|--------|----------|---------------|
| **Thonny** | کامل مبتدیان | [thonny.org](https://thonny.org) |
| **PyCharm** | توسعه حرفه‌ای | [jetbrains.com/pycharm](https://jetbrains.com/pycharm) |
| **Sublime Text** | ویرایش سریع و سبک | [sublimetext.com](https://sublimetext.com) |

## ۳. استفاده از ترمینال/خط فرمان

ترمینال (همچنین خط فرمان، کنسول یا shell نامیده می‌شود) جایی است که برای اجرای برنامه‌های خود دستورات را تایپ می‌کنید.

### باز کردن ترمینال

- **ویندوز:** `Win + R` را فشار دهید، `cmd` را تایپ کنید، Enter را بزنید
  - یا از PowerShell استفاده کنید (مدرن‌تر)
- **مک:** `Cmd + Space` را فشار دهید، `terminal` را تایپ کنید، Enter را بزنید
- **لینوکس:** `Ctrl + Alt + T` را فشار دهید

### دستورات پایه ترمینال

اینها به شما کمک می‌کنند به فایل‌های پایتون خود پیمایش کنید:

```bash
# ببینید در کدام پوشه هستید
pwd          # مک/لینوکس
cd           # ویندوز

# لیست فایل‌ها در پوشه فعلی
ls           # مک/لینوکس
dir          # ویندوز

# تغییر به یک پوشه دیگر
cd Documents
cd my_project

# برو به یک پوشه بالاتر
cd ..

# برو به پوشه خانگی
cd ~         # مک/لینوکس
cd %USERPROFILE%   # ویندوز
```

## اجرای اولین برنامه پایتون شما

### روش ۱: استفاده از IDLE

۱. IDLE را باز کنید
۲. روی **File → New File** کلیک کنید
۳. تایپ کنید: `print("Hello, World!")`
۴. روی **Run → Run Module** کلیک کنید (یا F5 را فشار دهید)
۵. هنگام درخواست فایل را ذخیره کنید (از پسوند `.py` استفاده کنید)

### روش ۲: استفاده از VS Code

۱. VS Code را باز کنید
۲. روی **File → New File** کلیک کنید
۳. آن را به عنوان `hello.py` ذخیره کنید
۴. تایپ کنید: `print("Hello, World!")`
۵. روی دکمه "Play" (▶) در بالا سمت راست کلیک کنید، یا راست‌کلیک کرده و "Run Python File" را انتخاب کنید

### روش ۳: استفاده از ترمینال

۱. یک فایل به نام `hello.py` در ویرایشگر خود بسازید
۲. این کد را اضافه کنید: `print("Hello, World!")`
۳. ترمینال را باز کنید
۴. با استفاده از `cd` به پوشه حاوی فایل خود بروید
۵. اجرا کنید: `python hello.py` (یا `python3 hello.py`)

## جریان کار کدنویسی شما

در اینجا فرآیند معمول برای نوشتن و اجرای کد پایتون آمده است:

```
۱. ویرایشگر کد خود را باز کنید
        ↓
۲. کد خود را بنویسید یا ویرایش کنید
        ↓
۳. فایل را ذخیره کنید (Ctrl+S / Cmd+S)
        ↓
۴. ترمینال را باز کنید
        ↓
۵. به پوشه فایل خود بروید
        ↓
۶. اجرا کنید: python filename.py
        ↓
۷. خروجی را ببینید!
        ↓
۸. برای تغییرات به مرحله ۲ برگردید
```

---

## اشتباهات رایج مبتدیان

### اشتباه ۱: ذخیره نکردن قبل از اجرا

**مشکل:** تغییراتی ایجاد می‌کنید، برنامه را اجرا می‌کنید، اما خروجی قدیمی را می‌بینید.

**راه‌حل:** همیشه قبل از اجرا فایل خود را ذخیره کنید (Ctrl+S / Cmd+S).

### اشتباه ۲: پسوند فایل اشتباه

**مشکل:** فایل شما `myprogram.txt` به جای `myprogram.py` نامیده شده.

**راه‌حل:** فایل‌های پایتون باید با `.py` تمام شوند.

### اشتباه ۳: پایتون در PATH نیست (ویندوز)

**مشکل:** `python` را تایپ می‌کنید اما خطایی می‌گیرید که شناخته نشده.

**راه‌حل:** در طول نصب پایتون، "Add Python to PATH" را علامت بزنید. اگر این را از دست دادید، پایتون را دوباره نصب کنید.

### اشتباه ۴: اجرای پایتون از پوشه اشتباه

**مشکل:** سعی می‌کنید `python hello.py` را اجرا کنید اما در پوشه اشتباه هستید.

**راه‌حل:** با استفاده از `cd` به پوشه حاوی فایل پایتون خود بروید.

## خودتان امتحان کنید: تمرین نصب

### تمرین ۱: تأیید تنظیمات شما

۱. ترمینال خود را باز کنید
۲. `python --version` (یا `python3 --version`) را تایپ کنید
۳. باید نسخه پایتون خود را ببینید
۴. برای ورود به حالت تعاملی `python` (یا `python3`) را تایپ کنید
۵. `print("It works!")` را تایپ کنید
۶. باید "It works!" را ببینید
۷. با تایپ `exit()` خارج شوید

### تمرین ۲: ایجاد و اجرای یک فایل

۱. ویرایشگر کد خود را باز کنید
۲. یک فایل جدید بسازید
۳. آن را به عنوان `test.py` ذخیره کنید
۴. تایپ کنید: `print("My first Python program!")`
۵. فایل را ذخیره کنید
۶. ترمینال را باز کنید و به محل فایل بروید
۷. اجرا کنید: `python test.py`
۸. باید خروجی را ببینید!

## مرجع سریع

### باز کردن ترمینال

| سیستم عامل | نحوه باز کردن |
|------------------|-------------|
| ویندوز | `Win + R` را فشار دهید، `cmd` را تایپ کنید |
| مک | `Cmd + Space` را فشار دهید، `terminal` را تایپ کنید |
| لینوکس | `Ctrl + Alt + T` را فشار دهید |

### اجرای پایتون

| کار | دستور |
|------|---------|
| بررسی نسخه | `python --version` |
| حالت تعاملی | `python` |
| اجرای یک فایل | `python filename.py` |

### دستورات رایج ترمینال

| کار | مک/لینوکس | ویندوز |
|------|-----------|---------|
| پوشه فعلی | `pwd` | `cd` |
| لیست فایل‌ها | `ls` | `dir` |
| تغییر پوشه | `cd foldername` | `cd foldername` |
| برو به بالا | `cd ..` | `cd ..` |
| برو به خانه | `cd ~` | `cd %USERPROFILE%` |

## نکات کلیدی

۱. **سه چیز نیاز دارید:** پایتون، یک ویرایشگر کد، و یک ترمینال
