# انواع داده در پایتون

## آنچه خواهید آموخت
- چند نوع مختلف داده در پایتون وجود دارد
- چگونه متوجه شوید که چه نوع داده‌ای با چه داده‌ای کار می‌کنید
- چه زمانی از هر نوع داده استفاده کنید

---

## انواع داده چیست؟

انواع داده در برنامه‌نویسی مثل دسته‌بندی‌های متفاوت برای ذخیره‌سازی هستند.

در زندگی واقعی:
- **اعداد** (برای محاسبات)
- **متن** (برای نام‌ها، پیام‌ها)
- **بله/خیر** (برای سوال‌ها)
- **لیست‌ها** (برای مجموعه چیزها)

در پایتون، ما اینها را به انواع داده تبدیل می‌کنیم:
- اعداد → `int` و `float`
- متن → `str` (رشته)
- بله/خیر → `bool` (بولی)
- لیست‌ها → `list`, `tuple`, `dict`

---

## انواع داده اصلی پایتون

### ۱. اعداد صحیح (int)

اعداد کامل بدون اعشار.

```python
age = 25
count = -5
year = 2024
```

**چه زمانی استفاده کنیم:**
- شمارش چیزها
- سن
- سال

```python
students = 30
print(students)        # 30
print(type(students))  # <class 'int'>
```

### ۲. اعداد اعشاری (float)

اعداد با اعشار.

```python
price = 19.99
height = 5.8
pi = 3.14159
```

**چه زمانی استفاده کنیم:**
- قیمت‌ها
- اندازه‌گیری‌ها
- درصدها

```python
temperature = 23.5
print(temperature)        # 23.5
print(type(temperature))  # <class 'float'>
```

### ۳. رشته‌ها (str)

متن - داده‌های کاراکتری.

```python
name = "Alice"
message = 'Hello, World!'
multiline = """This is
a multiline
string"""
```

**چه زمانی استفاده کنیم:**
- نام‌ها
- پیام‌ها
- هر چیز مبتنی بر متن

```python
greeting = "Hello"
print(greeting)        # Hello
print(type(greeting))  # <class 'str'>
```

### ۴. بولی (bool)

فقط دو مقدار: `True` یا `False`.

```python
is_student = True
has_license = False
is_raining = True
```

**چه زمانی استفاده کنیم:**
- سوال‌های بله/خیر
- فعال/غیرفعال کردن ویژگی‌ها
- تصمیم‌گیری‌ها

```python
is_happy = True
print(is_happy)        # True
print(type(is_happy))  # <class 'bool'>
```

---

## مجموعه‌های داده (انواع پیشرفته‌تر)

### ۵. لیست‌ها (list)

مجموعه مرتبی از موارد.

```python
fruits = ["apple", "banana", "cherry"]
numbers = [1, 2, 3, 4, 5]
mixed = [1, "hello", True, 3.14]
```

**چه زمانی استفاده کنیم:**
- مجموعه‌ای از چیزهای مشابه
- وقتی نیاز به تغییر دارید
- وقتی ترتیب مهم است

```python
colors = ["red", "green", "blue"]
print(colors)        # ['red', 'green', 'blue']
print(type(colors))  # <class 'list'>
```

### ۶. تاپل‌ها (tuple)

مانند لیست، اما **نمی‌توان تغییر داد**.

```python
coordinates = (10, 20)
days = ("Monday", "Tuesday", "Wednesday")
```

**چه زمانی استفاده کنیم:**
- موقعیت‌های ثابت
- تاریخ‌های تغییرناپذیر
- وقتی می‌خواهید مطمئن شوید داده تغییر نمی‌کند

```python
point = (5, 10)
print(point)        # (5, 10)
print(type(point))  # <class 'tuple'>
```

### ۷. دیکشنری‌ها (dict)

جفت‌های کلید-مقدار.

```python
person = {
    "name": "Alice",
    "age": 25,
    "city": "New York"
}
```

**چه زمانی استفاده کنیم:**
- داده‌های مرتبط با هم
- جستجو سریع با کلید
- نمایش اشیاء

```python
student = {"name": "Bob", "grade": "A"}
print(student)        # {'name': 'Bob', 'grade': 'A'}
print(type(student))  # <class 'dict'>
```

### ۸. مجموعه‌ها (set)

مجموعه‌ای از موارد منحصربفرد (بدون تکرار).

```python
unique_numbers = {1, 2, 3, 4, 5}
```

**چه زمانی استفاده کنیم:**
- وقتی می‌خواهید تکرارها را حذف کنید
- عملیات مجموعه (اتحاد، اشتراک)

```python
numbers = {1, 2, 2, 3, 3, 3}
print(numbers)        # {1, 2, 3}
print(type(numbers))  # <class 'set'>
```

---

## خلاصه انواع داده

| نوع | نام پایتون | مثال | استفاده اصلی |
|------|------------|-------|-------------|
| عدد صحیح | `int` | `42`, `-7` | شمارش |
| عدد اعشاری | `float` | `3.14`, `-0.5` | اندازه‌گیری‌ها |
| رشته | `str` | `"Hello"` | متن |
| بولی | `bool` | `True`, `False` | بله/خیر |
| لیست | `list` | `[1, 2, 3]` | مجموعه قابل تغییر |
| تاپل | `tuple` | `(1, 2, 3)` | مجموعه ثابت |
| دیکشنری | `dict` | `{"key": "value"}` | جفت کلید-مقدار |
| مجموعه | `set` | `{1, 2, 3}` | موارد منحصربفرد |

## بررسی نوع داده

می‌توانید از تابع `type()` استفاده کنید تا ببینید با چه نوع داده‌ای کار می‌کنید:

```python
print(type(42))          # <class 'int'>
print(type(3.14))        # <class 'float'>
print(type("Hello"))     # <class 'str'>
print(type(True))        # <class 'bool'>
print(type([1, 2, 3]))   # <class 'list'>
```

**بررسی نوع داده:**

```python
age = 25
print(type(age) == int)  # True

name = "Alice"
print(isinstance(name, str))  # True
```

## تبدیل انواع داده

گاهی اوقات نیاز دارید یک نوع داده را به نوع دیگر تبدیل کنید:

### تبدیل به رشته

```python
number = 42
text = str(number)
print(text)       # "42"
print(type(text)) # <class 'str'>
```

### تبدیل به عدد

```python
text = "100"
number = int(text)
print(number)     # 100
print(type(number)) # <class 'int'>

# برای اعشار
pi_text = "3.14"
pi_number = float(pi_text)
print(pi_number)  # 3.14
```

### تبدیل به لیست

```python
text = "Hello"
char_list = list(text)
print(char_list)  # ['H', 'e', 'l', 'l', 'o']
```

### تبدیل به تاپل

```python
my_list = [1, 2, 3]
my_tuple = tuple(my_list)
print(my_tuple)   # (1, 2, 3)
```

## اشتباهات رایج

### اشتباه ۱: جمع رشته با عدد

```python
# ❌ اشتباه
age = 25
text = "I am " + age + " years old"  # TypeError!

# ✅ صحیح
text = "I am " + str(age) + " years old"

# ✅ حتی بهتر
text = f"I am {age} years old"
```

### اشتباه ۲: سعی در تغییر یک تاپل

```python
# ❌ اشتباه
point = (10, 20)
point[0] = 30  # TypeError! تاپل‌ها نمی‌توانند تغییر کنند

# ✅ صحیح
point = [10, 20]  # از لیست استفاده کنید
point[0] = 30     # حالا کار می‌کند!
```

### اشتباه ۳: سعی برای دسترسی به کلیدی که وجود ندارد

```python
# ❌ اشتباه
person = {"name": "Alice"}
print(person["age"])  # KeyError!

# ✅ صحیح
print(person.get("age", "Unknown"))  # "Unknown"
```

## نکات کلیدی

۱. **استفاده از نوع داده درست** باعث کد تمیزتر و بدون باگ می‌شود
۲. **تابع `type()`** به شما می‌گوید با چه نوع داده‌ای کار می‌کنید
۳. **پایتون انواع داده را تشخیص می‌دهد** - نیازی به اعلان صریح نیست
۴. **با `str()`، `int()`، `float()`** می‌توانید انواع داده را تبدیل کنید
۵. **لیست‌ها قابل تغییر هستند**، **تاپل‌ها ثابت**، **دیکشنری‌ها از کلیدها استفاده می‌کنند**
