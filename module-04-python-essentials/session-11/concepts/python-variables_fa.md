# متغیرها در پایتون

## آنچه خواهید آموخت
- متغیرها چه هستند و چرا به آنها نیاز داریم
- چگونه متغیرها را در پایتون ایجاد و استفاده کنید
- قوانین نام‌گذاری متغیرها
- نکات و ترفندهای کار با متغیرها

---

## متغیرها چه هستند؟

متغیرها مثل **جعبه‌های برچسب‌دار** هستند که داده را نگه می‌دارند. به جای اینکه مقدار را مستقیماً استفاده کنید، می‌توانید آن را در یک متغیر ذخیره کنید و با نام آن به آن ارجاع دهید.

### تشبیه: جعبه‌های برچسب‌دار

```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   جعبه با برچسب   │    │   جعبه با برچسب   │    │   جعبه با برچسب   │
│    "name"       │    │     "age"       │    │    "price"      │
├─────────────────┤    ├─────────────────┤    ├─────────────────┤
│     "Alice"     │    │       25        │    │      9.99       │
└─────────────────┘    └─────────────────┘    └─────────────────┘
```

با متغیرها می‌توانید:
- داده را **ذخیره** کنید برای استفاده بعدی
- به جای مقادیر گمنام از **نام‌های معنی‌دار** استفاده کنید
- مقادیر را **تغییر** دهید و کد شما به طور خودکار به‌روز می‌شود

---

## ایجاد متغیرها (تخصیص)

در پایتون، متغیرها با **تخصیص** ایجاد می‌شوند:

```python
variable_name = value
```

### مثال‌های ساده

```python
# رشته (متن)
name = "Alice"

# عدد صحیح
age = 25

# عدد اعشاری
height = 5.6

# بولی (درست/نادرست)
is_student = True
```

پایتون **خودکار تشخیص می‌دهد** که چه نوع داده‌ای را ذخیره می‌کنید!

### خودتان امتحان کنید

```python
# یک متغیر ایجاد کنید
favorite_color = "blue"

# از آن استفاده کنید
print(favorite_color)

# یکی دیگر بسازید
year = 2024
print(year)
```

## استفاده از متغیرها

### نمایش مقادیر متغیر

```python
name = "Alice"
age = 25

print(name)              # Alice
print(age)               # 25
print("Name:", name)     # Name: Alice
print(f"{name} is {age} years old")  # Alice is 25 years old
```

### انجام عملیات با متغیرها

```python
# اعداد
x = 10
y = 5
sum_result = x + y
print(sum_result)        # 15

# رشته‌ها
first_name = "John"
last_name = "Doe"
full_name = first_name + " " + last_name
print(full_name)         # John Doe

# رشته‌ها و اعداد
name = "Alice"
age = 25
message = f"{name} is {age} years old"
print(message)           # Alice is 25 years old
```

## تغییر مقادیر متغیر

متغیرها می‌توانند **مقادیر جدید** بگیرند:

```python
score = 100
print(score)             # 100

score = 200              # مقدار جدید
print(score)             # 200

score = score + 50       # به مقدار فعلی اضافه کن
print(score)             # 250

score += 25              # راه کوتاه‌تر برای اضافه کردن
print(score)             # 275
```

### انواع داده مختلف در یک متغیر

متغیرها می‌توانند **انواع مختلفی** بگیرند (البته مراقب باشید!):

```python
# می‌توانید نوع داده را تغییر دهید
value = 42               # عدد صحیح
print(value)             # 42

value = "forty-two"      # حالا یک رشته
print(value)             # forty-two

value = True             # حالا یک بولی
print(value)             # True
```

⚠️ **هشدار:** اگرچه پایتون این را اجازه می‌دهد، معمولاً **خوب نیست** که نوع یک متغیر را تغییر دهید - باعث سردرگمی می‌شود!

## قوانین نام‌گذاری متغیرها

### ✅ قوانین اجباری (بایدها)

۱. فقط حروف، اعداد و آندرلاین (`_`)
   ```python
   name = "Alice"         # ✅
   name2 = "Bob"          # ✅
   user_name = "Charlie"  # ✅
   ```

۲. باید با یک حرف یا آندرلاین شروع شود
   ```python
   name = "Alice"         # ✅
   _name = "Bob"          # ✅
   2name = "Charlie"      # ❌ نمی‌تواند با عدد شروع شود
   ```

۳. حساس به بزرگی و کوچکی حروف
   ```python
   Name = "Alice"
   name = "Bob"
   # اینها دو متغیر متفاوت هستند!
   print(Name)            # Alice
   print(name)            # Bob
   ```

### ❌ کلمات رزرو شده (نمی‌توانید از اینها استفاده کنید)

این کلمات معنی خاصی در پایتون دارند:

```
False   await   else    import  pass
None    break   except  in      raise
True    class   finally is      return
and     continue for    lambda  try
as      def     from    nonlocal while
assert  del     global  not     with
async   elif    if      or      yield
```

**اشتباه:**
```python
class = "Math"           # ❌ 'class' رزرو شده است
def = 5                  # ❌ 'def' رزرو شده است
```

### ✅ قراردادهای خوب (بایدهای اختیاری)

۱. از حروف کوچک برای متغیرهای عادی استفاده کنید
   ```python
   name = "Alice"         # ✅
   Name = "Alice"         # ممکن است برای کار دیگری استفاده شود
   ```

۲. از آندرلاین برای جدا کردن کلمات استفاده کنید (snake_case)
   ```python
   user_name = "Alice"    # ✅
   userName = "Alice"     # کار می‌کند، اما در پایتون متداول نیست
   ```

۳. از نام‌های معنی‌دار استفاده کنید
   ```python
   x = "Alice"            # ❌ معنی‌دار نیست
   user_name = "Alice"    # ✅ واضح است!
   ```

۴. از نام‌های خیلی کوتاه خودداری کنید (مگر در حلقه‌ها)
   ```python
   n = "Alice"            # ❌ واضح نیست
   name = "Alice"         # ✅ خیلی بهتر
   ```

---

## سبک‌های نام‌گذاری

| سبک | استفاده | مثال |
|-------|---------|--------|
| `snake_case` | متغیرها، توابع | `user_name` |
| `camelCase` | گاهی در کد قدیمی | `userName` |
| `PascalCase` | کلاس‌ها | `UserAccount` |
| `SCREAMING_SNAKE_CASE` | ثابت‌ها | `MAX_SIZE` |

در پایتون:
- متغیرها و توابع → `snake_case`
- کلاس‌ها → `PascalCase`
- ثابت‌ها → `SCREAMING_SNAKE_CASE`

## چندین متغیر در یک خط

### تخصیص چندگانه

```python
# ایجاد چندین متغیر به طور همزمان
x, y, z = 1, 2, 3
print(x)                 # 1
print(y)                 # 2
print(z)                 # 3
```

### تخصیص یک مقدار به چندین متغیر

```python
# همه متغیرها یک مقدار می‌گیرند
a = b = c = 0
print(a)                 # 0
print(b)                 # 0
print(c)                 # 0
```

### جابه‌جایی متغیرها

```python
# جابه‌جایی متغیرها بدون متغیر کمکی
a = 5
b = 10
a, b = b, a
print(a)                 # 10
print(b)                 # 5
```

## حذف متغیرها

می‌توانید متغیرها را با `del` حذف کنید:

```python
name = "Alice"
print(name)              # Alice

del name                 # متغیر حذف شد
print(name)              # ❌ NameError! name دیگر وجود ندارد
```

⚠️ **احتیاط:** معمولاً نیازی به حذف متغیرها نیست - پایتون خودش به طور خودکار آنها را مدیریت می‌کند.

## متغیرهای خاص

### متغیرهای تک‌زیرخط (`_`)

```python
# _ معمولاً برای مقادیر اهمیت‌نداده استفاده می‌شود
for _ in range(5):
    print("Hello")

# یا برای نگه‌داشتن آخرین مقدار در حالت تعاملی
```

### متغیرهای دو زیرخط (`__name__`)

```python
# متغیرهای با دو زیرخط در هر طرف معمولاً برای اهداف خاص پایتون استفاده می‌شوند
print(__name__)            # __main__ (هنگام اجرای مستقیم)
```

## مرجع سریع

### ایجاد متغیرها

```python
# نحو پایه
name = value

# چندین متغیر
x, y, z = 1, 2, 3
a = b = c = 0
```

### قوانین نام‌گذاری

- ✅ حروف، اعداد، آندرلاین
- ✅ با حرف یا آندرلاین شروع شود
- ❌ نمی‌تواند با عدد شروع شود
- ❌ نمی‌تواند کلمات رزرو شده باشد
- ✅ از snake_case برای متغیرها استفاده کنید

### استفاده از متغیرها

```python
# چاپ
print(variable)
print(f"Text {variable}")

# عملیات
result = variable1 + variable2
```

## نکات کلیدی

۱. **متغیرها جعبه‌های برچسب‌دار** برای ذخیره داده هستند
۲. **تخصیص با `=`** انجام می‌شود: `name = value`
۳. **نام‌های معنی‌دار** انتخاب کنید - `user_name` بهتر از `x` است
۴. **snake_case** برای متغیرها در پایتون استفاده کنید
۵. **از کلمات رزرو شده** مانند `class`، `def`، `if` استفاده نکنید
۶. **متغیرها می‌توانند تغییر کنند** - مقادیر جدید را بگیرند

## گام بعدی چیست؟

- **انواع داده** - اعداد، رشته‌ها، لیست‌ها، و غیره
- **تبدیل نوع** - تغییر از یک نوع به نوع دیگر
