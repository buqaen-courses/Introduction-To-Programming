# تبدیل دهدهی-دودویی: یادگیری ترجمه بین اعداد انسان و کامپیوتر

## مقدمه: چرا تبدیل کنیم؟

تصور کنید به کشوری سفر می‌کنید که مردم آن متفاوت می‌شمارند. در یک کشور، برای ۱۲ عدد می‌گویند "دوازده". در کشور دیگر، ممکن است بگویند "یک-دو" یا از کلمات کاملاً متفاوتی استفاده کنند. به عنوان مسافر، باید یاد بگیرید ترجمه کنید!

**دهدهی (مبنای ۱۰)** نحوه شمارش انسان‌ها است (۰-۹).
**دودویی (مبنای ۲)** نحوه شمارش کامپیوترها است (۰-۱).

یادگیری تبدیل بین آنها مثل یادگیری صحبت کردن به هر دو زبان به طور روان است.

## تبدیل دهدهی به دودویی

### روش ۱: تفریق (دوستانه برای مبتدیان)

این روش شهودی است - مانند پرداخت با سکه‌های ارزش‌های مختلف.

**"سکه‌های" توان ۲**
```
موقعیت:   ۷    ۶    ۵    ۴    ۳    ۲    ۱    ۰
مقدار:   ۱۲۸   ۶۴   ۳۲   ۱۶    ۸    ۴    ۲    ۱
```

**فرآیند گام به گام:**
۱. بزرگترین توان ۲ که در عدد شما جا می‌شود را پیدا کنید
۲. ۱ در آن موقعیت قرار دهید
۳. آن مقدار را از عدد خود تفریق کنید
۴. تا رسیدن به ۰ تکرار کنید
۵. موقعیت‌های باقی‌مانده را با ۰ پر کنید

### مثال: تبدیل ۴۵ به دودویی

```
مرحله ۱: بزرگترین توان ۲ ≤ ۴۵ چیست؟
        ۶۴ بزرگتر است، ۳۲ جا می‌شود! ✓
        ۱ در موقعیت ۵ (۳۲) قرار می‌دهیم
        ۴۵ - ۳۲ = ۱۳ باقی‌مانده
        
        موقعیت‌ها:  ۷  ۶  ۵  ۴  ۳  ۲  ۱  ۰
        مقادیر:   ۱۲۸ ۶۴ ۳۲ ۱۶  ۸  ۴  ۲  ۱
        بیت‌ها:      ۰  ۰  ۱  ؟  ؟  ؟  ؟  ؟

مرحله ۲: بزرگترین توان ۲ ≤ ۱۳ چیست؟
        ۱۶ بزرگتر است، ۸ جا می‌شود! ✓
        ۱ در موقعیت ۳ (۸) قرار می‌دهیم
        ۱۳ - ۸ = ۵ باقی‌مانده
        
        بیت‌ها:      ۰  ۰  ۱  ۰  ۱  ؟  ؟  ؟

مرحله ۳: بزرگترین توان ۲ ≤ ۵ چیست؟
        ۸ بزرگتر است، ۴ جا می‌شود! ✓
        ۱ در موقعیت ۲ (۴) قرار می‌دهیم
        ۵ - ۴ = ۱ باقی‌مانده
        
        بیت‌ها:      ۰  ۰  ۱  ۰  ۱  ۱  ؟  ؟

مرحله ۴: بزرگترین توان ۲ ≤ ۱ چیست؟
        ۱ جا می‌شود! ✓
        ۱ در موقعیت ۰ (۱) قرار می‌دهیم
        ۱ - ۱ = ۰ (تمام!)
        
        بیت‌ها:      ۰  ۰  ۱  ۰  ۱  ۱  ۰  ۱

پاسخ: ۴۵₁₀ = ۰۰۱۰۱۱۰۱₂

بررسی: ۳۲ + ۸ + ۴ + ۱ = ۴۵ ✓
```

### روش ۲: تقسیم بر ۲ (الگوریتم سنتی)

**قاعده**: تقسیم بر ۲، باقی‌مانده‌ها را دنبال کنید، باقی‌مانده‌ها را از پایین به بالا بخوانید.

**مثال: تبدیل ۴۲ به دودویی**
```
۴۲ ÷ ۲ = ۲۱ باقی‌مانده ۰  ↑ (اینها را بالا بخوانید)
۲۱ ÷ ۲ = ۱۰ باقی‌مانده ۱  ↑
۱۰ ÷ ۲ = ۵  باقی‌مانده ۰  ↑
۵  ÷ ۲ = ۲  باقی‌مانده ۱  ↑
۲  ÷ ۲ = ۱  باقی‌مانده ۰  ↑
۱  ÷ ۲ = ۰  باقی‌مانده ۱  ↑ (از اینجا شروع کنید)

خواندن از پایین به بالا: ۱۰۱۰۱۰

پاسخ: ۴۲₁₀ = ۱۰۱۰۱۰₂

بررسی: ۳۲ + ۸ + ۲ = ۴۲ ✓
```

**چرا کار می‌کند**: هر تقسیم بر ۲ به ما می‌گوید عدد زوج است (باقی‌مانده ۰) یا فرد (باقی‌مانده ۱). این کمترین بیت معنی‌دار است. با خارج‌قسمت ادامه می‌دهیم تا ۰ شود.

## تبدیل دودویی به دهدهی

### روش استاندارد: جمع ارزش‌های مکانی

**فرمول ساده**: جایی که ۱ است، ارزش آن موقعیت را اضافه کنید.

### مثال: تبدیل ۱۰۱۱۰۱ به دهدهی

```
موقعیت:   ۵    ۴    ۳    ۲    ۱    ۰
بیت:        ۱    ۰    ۱    ۱    ۰    ۱
مقدار:     ۳۲   ۱۶    ۸    ۴    ۲    ۱
           │    │    │    │    │    │
         ۳۲×۱ + ۱۶×۰ + ۸×۱ + ۴×۱ + ۲×۰ + ۱×۱
           │    │    │    │    │    │
          ۳۲  +  ۰  +  ۸  +  ۴  +  ۰  +  ۱  = ۴۵

پاسخ: ۱۰۱۱۰۱₂ = ۴۵۱₀
```

### روش میانبر "دو برابر کردن"

این روش عدد دودویی را از چپ به راست پردازش می‌کند:

**الگوریتم**:
۱. با ۰ شروع کنید
۲. برای هر بیت: نتیجه فعلی را دو برابر کنید، سپس بیت را اضافه کنید
۳. نتیجه نهایی پاسخ است

**مثال: تبدیل ۱۰۱۱۰۱ به دهدهی**
```
بیت‌ها:  ۱    ۰    ۱    ۱    ۰    ۱

مرحله ۱: با ۰ شروع کنید
مرحله ۲: ۰ × ۲ + ۱ = ۱
مرحله ۳: ۱ × ۲ + ۰ = ۲
مرحله ۴: ۲ × ۲ + ۱ = ۵
مرحله ۵: ۵ × ۲ + ۱ = ۱۱
مرحله ۶: ۱۱ × ۲ + ۰ = ۲۲
مرحله ۷: ۲۲ × ۲ + ۱ = ۴۵

پاسخ: ۴۵۱₀
```

**چرا کار می‌کند**: این همان است که: ((((۱×۲+۰)×۲+۱)×۲+۱)×۲+۰)×۲+۱

## ترفندهای تبدیل سریع

### ترفند ۱: فرد یا زوج؟

به بیت راست‌ترین نگاه کنید:
- به **۰** ختم می‌شود = **زوج**
- به **۱** ختم می‌شود = **فرد**

**مثال‌ها:**
- ۱۰۱۰₂ به ۰ ختم می‌شود → زوج → ۱۰₁₀ ✓
- ۱۱۱۱₂ به ۱ ختم می‌شود → فرد → ۱۵₁₀ ✓

### ترفند ۲: آیا توان ۲ است؟

یک عدد توان ۲ است اگر دودویی آن دقیقاً **یک ۱** و بقیه ۰ داشته باشد:
- ۰۰۰۰۱۰۰۰ = ۸ = ۲³ ✓
- ۰۰۱۰۰۰۰۰ = ۳۲ = ۲⁵ ✓
- ۰۰۰۰۱۱۰۰ = ۱۲ (توان ۲ نیست، دو ۱ دارد) ✗

### ترفند ۳: الگوی جمع دودویی

وقتی ۱ را به عدد دودویی که به همه ۱ ختم می‌شود اضافه می‌کنیم:
```
   ۰۱۱۱  (۷)
 + ۰۰۰۱  (۱)
 ------
   ۱۰۰۰  (۸)
```

همه ۱های انتهایی به ۰ تبدیل می‌شوند، و اولین ۰ به ۱ تبدیل می‌شود!

## نکات کلیدی

۱. **دو روش تبدیل اصلی:**
   - دهدهی → دودویی: تفریق توان‌های ۲ (آسان‌ترین) یا تقسیم بر ۲
   - دودویی → دهدهی: جمع ارزش‌های مکانی یا روش دو برابر کردن

۲. **توان‌های ۲ را حفظ کنید**: آنها در محاسبات همه جا هستند

۳. **۸ بیت = ۱ بایت**: واحد ذخیره‌سازی بنیادین

۴. **الگوها به تشخیص کمک می‌کنند:**
   - اعداد زوج به ۰ ختم می‌شوند
   - توان‌های ۲ فقط یک ۱ و بقیه ۰ دارند
   - ۲۵۵ = ۱۱۱۱۱۱۱۱ (همه بیت‌ها روشن)

۵. **تمرین کامل می‌کند**: با اعداد کوچک شروع کنید، بالا بروید
