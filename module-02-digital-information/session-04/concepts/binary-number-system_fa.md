# سیستم اعداد دودویی: کامپیوترها چگونه با فقط دو رقم می‌شمارند

## مقدمه: چرا کامپیوترها فقط از ۰ و ۱ استفاده می‌کنند؟

تصور کنید باید با فقط دو دستتان بشمارید - نه ده انگشت، فقط "دست بالا" یا "دست پایین." کند بود، درست است؟ اما اگر میلیاردها دست داشتید که می‌توانستند فوراً بالا یا پایین بروند چطور؟ دقیقاً اینگونه است که کامپیوترها کار می‌کنند!

کامپیوترها از **سیستم اعداد دودویی** استفاده می‌کنند چون:
- آنها از سوئیچ‌های الکترونیکی (ترانزیستورها) ساخته شده‌اند
- هر سوئیچ فقط می‌تواند روشن یا خاموش باشد
- روشن = ۱، خاموش = ۰
- این سیستم دو حالته بسیار قابل اعتماد است

## تبدیل دهدهی به دودویی

### روش ۱: تفریق (آسان برای مبتدیان)

**مثال: تبدیل ۴۵ به دودویی**
